\doxysubsection{Detailed Description}
\mbox{\hyperlink{classchess_1_1pieces_1_1_orbit}{Orbit}} Class is a custom chess piece class. It allows movements from the piece to any location on the board within the mathematical radius of the piece. The default allowed radius is set to 3 tiles, but can be changed. The piece can jump to any square within its radius.
